# 互联网医院平台介绍 java935

## 项目概述

本项目是一款集患者移动端APP、医生移动端APP和后台管理平台于一体的互联网医院平台。它提供了在线问诊、预约挂号、快速处方、患者管理等多种功能，实现了医患之间的无缝对接，提升了医疗服务的便捷性与效率。

## 技术栈

项目采用了Axure进行高保真原型设计，具体技术实现未在简介中提及，以下为功能模块介绍。

## 功能模块

### 患者移动端APP

- **问诊功能**
  - 患者可按科室、疾病、医院等方式查找医生并进行在线问诊。
- **挂号功能**
  - 支持按疾病、医院在线预约挂号。
- **开药功能**
  - 复诊开药，支持电子处方流转。
  - 患者可在线购药或凭处方二维码到附近药店购药。
- **极速问诊**
- **疾病科普**
  - 提供疾病相关知识的图文介绍。

### 医生移动端APP

- **注册与登录**
  - 医生需上传“医生执业证书”和“职称证书”完成实名验证。
- **问诊功能**
- **开药问诊**
  - 对患者进行视频问诊接诊并开具处方。
- **挂号预约**
  - 查看患者门诊预约情况。
- **快速处方**
  - 编辑开处方信息，便于快速为患者开方。
- **患者管理**
  - 对患者来源进行分类，支持群发消息。
- **服务设置**
  - 开通平台服务。
- **个人信息管理**
  - 设置个人信息，查看收入情况。

### 管理后台(Web)端

- 包含医生管理、用户管理、处方管理、订单管理、财务管理等后台业务管理功能。
- 提供商品管理、运营管理、内容管理、交易统计、用户统计、医生统计等全面业务支持。

## 系统角色

- 管理后台操作员
